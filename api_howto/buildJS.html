Building an Application

Architecture of an Application using the Etsy API

First, let's create the files that we need for this. Right now, we only need an html page that will link to a file where our Javascript will be writen:

  • shop.html
  • shop.js

This is of course pretty minimal- in order to make your page more sophisticated you would of course write a CSS stylesheet or use something like Bootstrap, but for now we will focus on the API.


Building the HTML Page

I find the Etsy API is best accessible through jQuery, in which case jQuery will need to be loaded into the <head> tag of the html page. The link that I found used in examples on the Etsy documentation page were failing for being outdated, so loading from Google will work fine for us:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>

In particular, jQuery will be used to perform the Ajax requests on the API. For further documentation on this, look here!

Besides that, the HTML page must link to the shop.js file in the <body> tag, and it also must contain a div or nav tag with a unique id (let's call it shop) which the Javascript can manipulate.

<html>
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
</head>
<body>
  <div id="shop"></div>
  <sript type="text/javascript" src="test.js"></script>
</body>
</html>
