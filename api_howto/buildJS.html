Architecture of an Application using the Etsy API

In the case of the Etsy API, the first thing that we need will be contained in the <head> tag in order to use jQuery: 

<script type="text/javascript" src="http://code.jquery.com/jquery-1.4.2.min.js"></script>

Then, another <script> tag handles the rest of the functionality. Since right now our application is mostly static, as there are no searches being made by the user that will trigger an event, the script will have an outer shell that will look like this: 

(function($){
  $(document).ready(function(){

    // API calls will go here!

  });
})(jQuery);

The first two lines within the inner function are necessary in order to connect to the API: 

api_key = "YOUR-API-KEY";
terms = $('#etsy-terms').val();

In the case of this example, I will be using the keystring yytys7tcm73htzzb4f7npm70. You should go ahead and use your own! We will not be using the shared secret key.
